1 - Para compilar o programa, digite "g++ -o celular celular.cpp $(GLLIBS)" no temrinal na pasta root.
2 - Depois, execute digitando "./celular".

Instruções de uso:

Após a inicialização do programa, temos as seguintes teclas disponíveis para sua execução, 
os elementos surgem onde o ponteiro do mouse está localizado, é preciso clicar e arrastar para criar os elementos então segurando uma das seguintes teclas:

As teclas “S”, “W” e “E” selecionam os elementos que serão gerados no mundo, são eles Areia, Água e Madeira respectivamente;
As teclas “I”, “K”, “L”, “J”, “P” , “N” são para a rotação do ângulo de visão;
A Tecla “.” altera a iluminação global, em toggle, alternando entre noite e dia.
